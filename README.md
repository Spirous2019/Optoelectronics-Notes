# Optoelectronics Study Notes

Welcome to my comprehensive study notes for **Optoelectronics** (Spring 2026). This repository contains the complete LaTeX source code, figures, and compiled documents detailing the fundamental concepts and derivations of optoelectronic systems.

## 📖 Repository Structure

*   **/Complete_Study_Notes**: The core directory containing the main LaTeX document.
    *   **Preamble.tex**: Contains the custom formatting, packages, and macro definitions (like `keyinsight` and `keyresult` boxes).
    *   **/Chapters**: Individual `.tex` files for modular reading and compilation.
    *   **/Figures**: Categorized directories filled with the diagrams and graphs used to visualize the concepts.
    *   **Optoelectronics.pdf**: The compiled, ready-to-read final PDF of the notes.
*   **/Plots**: Custom Python scripts used to generate the precision, publication-quality mathematical plots embedded in the document.

## 🚀 Building the PDF

To compile the study notes locally, ensure you have a full TeX distribution (such as TeX Live or MiKTeX) installed. 

1. Navigate to the `Complete_Study_Notes` directory.
2. Compile the main document (typically using `pdflatex` or `latexmk`).
   
```bash
cd Complete_Study_Notes
pdflatex Optoelectronics.tex
```

## 📝 About the Content
These notes place a strong emphasis on maintaining a rigorous mathematical foundation without sacrificing clear physical intuition. Custom formatting environments have been used to highlight critical physical insights and final equations.
